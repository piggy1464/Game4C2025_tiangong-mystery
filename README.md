# Game4C2025_tiangong-mystery

# 物鹿山河·天工解密

> 2025 中国大学生计算机设计大赛参赛作品 | 参赛编号：2025002106 | 物理主题

作者均来自西南林业大学

注：仅供学习使用。 


## 📖 作品简介

《物鹿山河·天工解密》是一款融合中国传统物理文化元素的解谜类游戏，以"天工开物"为创作灵感，分为3个关卡，玩家在山河意境中探索古代匠造之美，通过机关解谜推动剧情发展。

游戏基于 **Unity** 引擎开发，涵盖场景设计、玩法设计、原创绘画美术、音乐音效及完整交互逻辑，是一次将传统文化与现代游戏技术结合的实践探索。

作品链接：https://www.bilibili.com/video/BV1trVozZEdc/?spm_id_from=333.1387.0.0&vd_source=c01a04ec15e86dfe4ce8f552f1838ef4

该项目获得计算机设计大赛国赛二等奖，省赛一等奖。（奖状见下）

## 🎮 运行说明

1. 下载 `release/` 目录下的可执行文件夹
2. 双击运行 `Our project.exe`
3. 推荐分辨率：**1920 × 1080**（游戏窗口默认以此分辨率运行，若屏幕分辨率更高窗口会相对较小，属正常现象）
4. 系统要求：Windows 10/11均可，64位

## 📁 目录说明
├── docs/ 设计与开发文档、官方文件
├── release/ 可执行程序（exe 打包）、答辩 PPT、路演视频 、部分素材、导入的unitypackage
